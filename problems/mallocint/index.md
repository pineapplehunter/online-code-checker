# mallocint

`n` と `n` 個の整数 `a_1, a_2, ..., a_n` が与えられるので、それぞれを 2 倍した値とその合計を求めてください。

* ヒント

* `malloc` 関数を使って、`n` 個の整数を格納するためのメモリを確保しましょう。
* `for` 文を使って、配列の各要素を 2 倍しましょう。
* `for` 文を使って、配列の各要素を合計しましょう。

## 入力例 1

```
3
1 2 3
```

## 出力例 1

```
2 4 6
12
```

## 入力例 2

```
5
2 3 7 11 13
```

## 出力例 2

```
4 6 14 22 26
72
```

## 入力例 3

```
1
10
```

## 出力例 3

```
20
20
```
